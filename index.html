    • Intro to PHP

    What is PHP?

  • PHP is an acronym for "PHP: Hypertext Preprocessor"
  • PHP is a widely-used, open source scripting language
  • PHP scripts are executed on the server
  • PHP is free to download and use

    • But why use PHP?

  • PHP runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  • PHP is compatible with almost all servers used today (Apache, IIS, etc.)
  • PHP supports a wide range of databases
  • PHP is free. Download it from the official PHP resource: www.php.net
  • PHP 7 supports new operators (like the spaceship operator: <=>)

    1.2 Comments

    Writing comments in php is really easy! all there is to it is for a single line comment, just like most languages its (//) or you can do (#) then the text, for multiple line comments all it is(/*) your multiple line comment text then (*/). Easy as that!


    1.3 Displaying Text

    There are multiple ways to display text in PHP, the most common way is saying (echo) then the text you want to display in quotes (" "), after that you need a semicolon.
    Almost all statements in PHP end in a semicolon (;). When you type (echo) it is in not case sensitive so you could also type (Echo), either works! It doesnt matter what type of quote is used (' ') or (" ")
    Now you can also implement html into strings in an echo statement, for example if i wanted to print a header i can do (echo "< h1 >Text< /h1 >";) and that will print a header.
    Same goes for (< li > Text < /li >) and (< p > Text < /p >)

    1.4 Declaring And Using Variables

    Just like displaying text, declaring and calling variables are really simple, there are multiple variable types you can store in a variable. For example:

  • String
  • Integer
  • Float
  • Boolean
  • Array
  • Object
  • NULL
  • Resource

    • To declare most variables is as simple as saying ($) then the variable name, for example ($name) then what its equal to, ($name = "john";) and they will also always end in a semicolon.
    This applies for most to all data types, so if i wanted to declare an interger i would do ($age = 35;) and for a boolean i would do ($isAwsome = true;), etc.
    Now if i wanted to print my variable i would use (echo) then the name of the variable starting with ($), so i can do (echo $name;) and it will print john.
    I can use concatenation with variables to so if i wanted to print (My name is john) i can do (echo "My name is ") then a (.) to add something then i can do ($name;) all together its (echo "My name is " . $name;) and the outpu0t of that is (My name is john).
    If i wanted to print (john is 35 years old) i can do, (echo $name . " is " . $age . " years old";).
